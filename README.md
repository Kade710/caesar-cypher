# Wise Crack

Wise Crack is a simple Python tool for working with basic ciphers. It helps you encrypt, decrypt, and break simple codes like the Caesar cipher and ROT13.

This project is made for learning how basic cryptography works and how simple ciphers can be broken using brute force.

---

## What it can do

* Caesar Cipher Encryption
* Caesar Cipher Decryption
* Caesar Cipher Brute Force (tries all 26 shifts)
* ROT13 Encoding/Decoding

---

## How to run it

Make sure you have Python 3 installed.

Then run:

```bash
python3 main.py
```

---

## How to use it

When you run the program, you will see a menu:

1. Caesar Encrypt
2. Caesar Decrypt
3. Caesar Brute Force
4. ROT13
5. Exit

Just type the number of the option you want and press enter.

---

## Example

### Caesar Encrypt

Input:

```
HELLO
Shift: 3
```

Output:

```
KHOOR
```

### Brute Force

Input:

```
EBZAM LNUFBM VATLX KXGXP
```

The program will try all shifts and show the readable message.
